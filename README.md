# Steam Ban Checker
## Requirements
* node JS
* npm
* Steam API key
## installation
* Download Repo
* open CMD in folder
* add your API key to apiKey.txt
* add your accounts to accounts.txt
* run ```npm install```
* run ```node index.js```
## Usage
* after installing, add or remove accounts in accounts.txt
* run Steam Ban Checker by double clicking run.bat
